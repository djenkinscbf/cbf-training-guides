Overview

What is Zoho WorkDrive?

Zoho WorkDrive is CBF’s cloud file platform, part of Zoho One. It stores documents centrally, organises them into shared Team Folders, and keeps a full history so nothing is lost.

How CBF uses it

Teams use WorkDrive to store and co-edit files, share them internally and externally with controlled permissions, and recover earlier versions when needed. It integrates closely with the rest of Zoho and the Office suite editors.

Getting Started

Accessing Zoho WorkDrive

Open WorkDrive in a browser to reach your dashboard of Team Folders and files.

1
Sign in to WorkDrive
  1. 1
    Go to workdrive.zoho.com (or open WorkDrive from Zoho One)
  2. 2
    Click Sign In and authenticate with your CBF account
  3. 3
    You are taken to the WorkDrive dashboard
Signing in at CBF: CBF accesses Zoho through Zoho One using single sign-on. Use your CBF account and complete two-factor authentication if prompted. Do not use a personal Zoho account for CBF work.
Organise

Creating and managing Team Folders

Team Folders are shared workspaces where a group keeps its files together, with access set per member.

2
Create and manage a Team Folder
  1. 1
    Click Team Folders in the left sidebar, then Create Team Folder
  2. 2
    Enter a name and description
  3. 3
    Add members and set their access level
  4. 4
    To change access later, open the folder, click the Settings icon, and adjust members and roles
Roles: WorkDrive offers granular Team Folder roles — commonly Admin, Organizer, Editor, Commenter, and Viewer — so you can give each person exactly the access they need.
Files

Uploading and organising files

Get files into WorkDrive and keep them tidy so they are easy to find later.

3
Upload and organise
  1. 1
    Click Upload in the toolbar, or drag and drop files in
  2. 2
    Use folders to group related files
  3. 3
    Rename files for clarity and add tags to make them searchable
Share

Sharing files and setting permissions

Share inside CBF with set permissions, or share externally with links you can lock down.

4
Share internally and externally
  1. 1
    To share internally, click Share, add colleagues, and set View, Edit, or Comment access
  2. 2
    To share externally, generate a public link
  3. 3
    Add an expiry date and password, and restrict downloads where needed
External sharing: For a remote team and outside partners, always set an expiry and, for anything sensitive, a password and download restriction on public links.
Recover

Version control and file recovery

WorkDrive keeps earlier versions and a trash bin, so mistakes and deletions are recoverable.

5
Restore a version or deleted file
  1. 1
    Right-click a file and choose Version History to view, restore, or download an earlier version
  2. 2
    To recover a deleted file, open Trash, select the file, and click Restore
Useful Features

Features worth knowing

Recent WorkDrive capabilities, including ones that suit a fully remote team.

TrueSync desktop app

TrueSync makes your WorkDrive files appear in a folder on your computer and downloads them only when opened, so you get desktop access without filling your drive.

Mobile apps

Access, preview, and share files from the WorkDrive mobile apps on Android and iOS.

Controlled external sharing

Public links support expiry dates, passwords, and download restrictions, important when sharing outside CBF.

Security and audit

Enable two-factor authentication and use activity logs to monitor file access and changes; data classification helps protect sensitive content.

Customisable file views

Add and sort columns such as owner, modified date, size, or type to scan large folders faster.

WorkDrive 6.0

The latest version brings AI-assisted, intelligent content management to help organise and surface information across your files.
